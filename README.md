# Бюджетирование

Спецификация инструмента быстрого пересчёта сценариев бюджета для строительных объектов.

## Репозиторий

- **Проект:** Бюджетирование (`byudzhetirovanie`)
- **Module registry:** [vladimirmurach-vibe/module-registry](https://github.com/vladimirmurach-vibe/module-registry)
- **Версия спецификации:** 0.2.1 (draft)

## Назначение

Замена трудоёмкой Excel-модели веб-системой для:
- расчёта денежных потоков и финансового результата по строительным объектам;
- создания сценариев бюджета/прогноза (все объекты или один);
- оперативного пересчёта (≤ 20 минут) и сравнения версий;
- консолидации на горизонте планирования;
- автоматической загрузки факта и формирования отчётов PDF/Excel.

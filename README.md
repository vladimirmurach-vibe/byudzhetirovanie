# Бюджетирование

Спецификация проекта системы корпоративного бюджетирования.

## Репозиторий

- **Проект:** Бюджетирование (`byudzhetirovanie`)
- **Module registry:** [vladimirmurach-vibe/module-registry](https://github.com/vladimirmurach-vibe/module-registry)
- **Версия спецификации:** 0.1.0 (draft)

## Структура `spec/`

| Файл | Назначение |
|------|-----------|
| `index.yaml` | Метаданные проекта и ссылки |
| `business.yaml` | Бизнес-контекст, роли, ограничения |
| `data.yaml` | Сущности и атрибуты |
| `requirements.yaml` | Функциональные требования |
| `ui.yaml` | Экраны и навигация |
| `modules.yaml` | Выбранные модули из реестра |
| `architecture.yaml` | Архитектурные решения |
| `acceptance.yaml` | Приёмочные тесты |
| `assumptions.yaml` | Допущения |
| `gaps.yaml` | Неопределённости |
| `validation.yaml` | Правила валидации |
| `decisions.yaml` | Архитектурные решения |
| `history.yaml` | История версий спецификации |
